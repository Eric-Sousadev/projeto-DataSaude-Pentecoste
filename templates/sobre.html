Sobre o Projeto

O DataSaúde nasceu com o objetivo de transformar dados de saúde pública em informação útil, acessível e visual. O foco é apoiar decisões, identificar riscos epidemiológicos e facilitar o acompanhamento de casos de arboviroses, internações, mortalidade e outros indicadores essenciais para Pentecoste.


Funcionalidades Principais

  • Dashboard Interativo: Visualize dados epidemiológicos através de gráficos dinâmicos e mapas interativos.
  • Tabela de Doenças: Acesse uma tabela detalhada com informações atualizadas sobre diversas doenças.
  • Campanhas de Saúde: Fique informado sobre campanhas de vacinação e prevenção promovidas pela Secretaria de Saúde.

Tecnologias Utilizadas

O sistema foi desenvolvido utilizando Flask para o backend, com HTML, CSS e JavaScript no frontend. Bibliotecas como Chart.js foram empregadas para a criação de gráficos interativos, proporcionando uma experiência visual rica e informativa.

Equipe de Desenvolvimento

Este projeto foi desenvolvido por uma equipe dedicada desenvolvedores web , todos comprometidos em melhorar a gestão da saúde em Pentecoste através da tecnologia.


Este sistema foi desenvolvido para auxiliar na análise das doenças recorrentes de Pentecoste, permitindo tomada de decisão mais rápida e eficiente.


Sugestões, melhorias ou parcerias podem ser enviadas através do e-mail: email@exemplo.com O objetivo é evoluir o DataSaúde continuamente e torná-lo cada vez mais útil para o município.
